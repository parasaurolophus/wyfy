WyFy

Android app that allows a device to connect to a wi fi network using credentials stored in a QR code or NFC tag.

Features

  • Enter wi fi parameters in the UI (SSID, security protocol, password etc.)

  • Write those parameters to a NDEF-compatible NFC tag

  • Share those parameters as QR code image

  • Connect to a wi fi network automatically by tapping a NFC tag created by this app

Note that the contents of the QR code images is compatible with that used for storing wi fi parameters by popular QR code scanning apps like Barcode Scanner, QR Droid and so on. Use any such app to scan QR codes created by this one.

The contents of NFC tags created by this app are NDEF "U" records with URI's in the identical format to that used in the QR codes. This allows both for maximum code re-use (smaller size, fewer bugs) and also maximizes potential inter-operability with other apps.

Why this app?

There are plenty of of apps in the Play store with similar functionality and more elaborate feature sets. You might consider using this one because:

  • It is small and lightweight, focused exactly on its intended purpose

  • It requests no permissions beyond those it actually needs for its user-visible functionality (NFC, wi fi settings)

  • It is completely ad free, generates no notification-bar spam or the like

  • It is open-source, offering the opportunity not only to verify that it does not contain any malware but also to use it as an example of how to accomplish a number of Android app programming tasks (app life-cycle, sharing via intents and content providers, NFC etc.)
